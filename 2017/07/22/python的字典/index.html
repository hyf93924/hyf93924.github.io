OK啦，测试通过啦，这就是python字典的使用，那么在什么情况下使用字典呢：

  • 你需要检索的内容有个唯一标识key，那样就可以用字典很轻易的获取
  • 你想通过一个key来增删一个元素
  • 你不需要是有序的，因为字典里的元素是无序的

一句话就是：如果你用一个非数字的key就用字典，如果需要有序，那就用list

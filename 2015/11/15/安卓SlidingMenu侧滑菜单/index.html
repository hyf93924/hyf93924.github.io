安卓SlidingMenu侧滑菜单

Nov 15, 2015
SlidingMenu侧滑菜单需要开源依赖包，依赖的是SlidingMenu和ActionBarSherlock，这些包可以在github上下载。

步骤：

  • 解压下载的开源包，将解压后SlidingMenu和ActionBarSherlockADT中，查看这两个包的属性，属性为lib，其实就是当做依赖包来使用的。
  • 创建自己的demo项目，将这两个包作为依赖包添加到项目中

  • 创建一个布局文件用作左侧菜单，但是布局中为：com.jeremyfeinstein.slidingmenu.lib.SlidingMenu,布局中加入一个按钮用作测试

  • 主布局不需要做改变，然后到主文件中做相应的修改：
    1.做的就是实例化SlidingMenu，设置模式，设置触摸点，并将SlidingMenu附加到activity上
    2.相应的代码：

    4.简单的SlidingMenu侧滑菜单MainActivity上的修改就完成了，这是简单的侧滑样式，想要特效啥的可以自己修改修改，完整的代码： 

    public class MainActivity extends Activity {
  private SlidingMenu slidingMenu;
  @Override
  protected void onCreate(Bundle savedInstanceState) {
      super.onCreate(savedInstanceState);
      setContentView(R.layout.activity_main);
      slidingMenu = new SlidingMenu(this);
      //设置slidingMenu模式
      slidingMenu.setMode(slidingMenu.LEFT);
      //设置宽度
      slidingMenu.setBehindOffsetRes(R.dimen.sliding_menu_offset);
      //设置全屏触摸
      slidingMenu.setTouchModeAbove(SlidingMenu.TOUCHMODE_FULLSCREEN);
      //将slidingMenu附加在activity上
      slidingMenu.attachToActivity(this, slidingMenu.SLIDING_CONTENT);
      slidingMenu.setMenu(R.layout.slidingmenu);
  }
  //通过物理菜单打开slidingmenu
  @Override
  public boolean onKeyDown(int keyCode, KeyEvent event) {
      switch (keyCode) {
      case KeyEvent.KEYCODE_MENU:
          slidingMenu.toggle(true);
          break;
      default:
          break;
      }
      return super.onKeyDown(keyCode, event);
  }
}

这个可以说是自己第一次使用开源的库进行开发，做完这个侧滑菜单发现使用开源库开发SlidingMenu是比较简单的，还有一个DrawerLayout的侧滑布局，不需要用SlidingMenu依赖包，下次试试DrawerLayout。
